## OBJECTIVE
Create a complete, single-page landing page for a mobile game called "Zigzag". The goal is to make the game look incredibly fun and encourage users to download it. The page should be modern, visually appealing, and fully responsive for both desktop and mobile.

## GAME CONCEPT
"Zigzag" is an online, international version of the "Telephone Game" (le téléphone arabe).
1.  A player starts a "chain" with a piece of text, a drawing, or a voice message.
2.  This message is passed to 6 other random players around the world, one by one.
3.  Each player only sees the previous step and must reinterpret it, alternating between drawing, writing a description, or recording a voice note.
4.  The core fun comes from the hilarious misinterpretations and the final reveal that shows the absurd transformation of the original message.
