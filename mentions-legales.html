Mentions légales

Ces informations ont pour but de vous expliquer qui est derrière Zigzag, comment le site est hébergé et comment vos données sont traitées.

1. Éditeur du site

Le site Zigzag.fun (site de démonstration) est édité par l’équipe Zigzag, projet en phase bêta, sans statut juridique définitif à ce stade.

Contact principal : Zigzag.contactus@gmail.com

2. Hébergeur

Le site est hébergé auprès d’un prestataire cloud européen ou américain (type OVH, Render, Vercel ou équivalent). Les données techniques (logs, métriques) sont conservées pour assurer le bon fonctionnement du service.

3. Propriété intellectuelle

L’ensemble des éléments graphiques, logos, illustrations, textes et contenus présents sur le site Zigzag sont protégés par le droit d’auteur et/ou le droit des marques. Toute reproduction ou réutilisation non autorisée est interdite.

4. Données personnelles

Zigzag collecte uniquement les données strictement nécessaires au fonctionnement du service, par exemple : adresse email pour la liste d’attente ou la newsletter.

Ces données sont stockées dans notre base de données (par exemple Supabase) et ne sont ni revendues ni utilisées à des fins publicitaires externes.

Conformément au RGPD, vous pouvez demander l’accès, la modification ou la suppression de vos données en nous écrivant à : Zigzag.contactus@gmail.com.

5. Cookies et mesure d’audience

Zigzag peut utiliser des outils de mesure d’audience simples (type analytics) afin de comprendre comment le site est utilisé (pages vues, temps passé, etc.). Ces données sont anonymisées autant que possible et utilisées uniquement pour améliorer l’expérience.

6. Responsabilité

Zigzag est un jeu communautaire. Les contenus générés par les utilisateurs (phrases, dessins, interprétations) relèvent de leur seule responsabilité. L’équipe se réserve le droit de modérer ou supprimer tout contenu manifestement illicite ou contraire aux valeurs du service.
